TouchPoint Class

Representation of a finger within TouchScript.

An object implementing this interface is created when user touches the screen. A unique id is assigned to it which doesn't change throughout its life.

Attention! Do not store references to these objects beyond touch's lifetime (i.e. when target finger is lifted off). These objects may be reused internally. Store unique ids instead.
